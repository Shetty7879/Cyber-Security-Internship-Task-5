## Task Title
Malware Types & Behavior Analysis (Basic)

## Objective
To understand different types of malware, analyze their behavior using VirusTotal, and learn basic malware prevention techniques.

## Overview
Malware is malicious software designed to damage systems, steal data, or disrupt operations. This task focuses on identifying common malware types and analyzing a known malware sample safely using VirusTotal.

## Malware Types Covered
- Virus
- Worm
- Trojan
- Ransomware

## Tool Used
- VirusTotal

## Analysis Performed
- Analyzed a known malware hash using VirusTotal
- Observed detection results from multiple antivirus engines
- Studied common malware behavior indicators such as file modification and network communication
- Understood the malware lifecycle and spread methods

## Malware Behavior Indicators
- File modification activity
- Network communication with external servers
- Detection by multiple antivirus engines

## Malware Spread Methods
- Email attachments and phishing emails
- Malicious websites and links
- Infected or pirated software downloads
- Removable media such as USB drives

## Malware Prevention Methods
- Use updated antivirus software
- Avoid unknown email attachments and links
- Keep operating systems and applications updated
- Avoid pirated or untrusted software
- Enable firewall and security settings
